// Текстовое описание алгоритма
v1
1. Вывести число N.
2. Инициализировать сумму (sum = 0)
3. Перебрать числа от 1 до N
4. Добавить каждое значение в sum
5. выводим результат - сумму чисел

v2
1. Получение данных.
2. Инициализация переменной: создать переменную (сум = 0) для хранения суммы.
3. вычисление суммы:
   3.1. использовать цикл для перебора всех чисел от 1 до N.
   3.2. Добавлять перебираемые значения к сумме
4. Вывод результата.

v3
1. Ввести число N
2. Создать переменную в которой будем хранить сумму (Например: sum = 0)
3. Пройтись циклом от 1 до N
4. Добавлять на каждом шаге текущее число к sum
5. По завершению вывести значение sum
