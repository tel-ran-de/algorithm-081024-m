// Задача: Вычислить сумму всех целых чисел от 1 до заданного числа N.

// Текстовое описание алгоритма
 1. Вычисление числа N
 2.Создать переменную где будем хранить сумму
 2.Пройтись по массиву чисел и посмотрень есть ли Не целые числа 
 3. Если все числа целые тогда сумируем и записываем в нашу созданую переменную
 4. Если нет, тогда сначала выводим массыв только целых чисел и тогда суммируем

// Решение
 int n = 6
 sum = 0

 for(int i = 1; i<= n; i++){
    sum += i
 }
 console.log(sum)
